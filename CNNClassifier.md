# Convolutional Neural Network (CNN) Classifier

This project implements a Convolutional Neural Network (CNN) for image classification tasks using TensorFlow/Keras.  
It demonstrates how CNNs automatically extract image features and improve classification accuracy compared to traditional machine learning methods.

---

# What's Covered

## Convolutional Neural Network (CNN)
- Convolutional Layers
- Feature Extraction
- Pooling Operations
- Fully Connected Layers

## Data Preprocessing
- Image normalization
- Reshaping image dimensions
- One-hot encoding of labels

## CNN Architecture
- Conv2D layers
- ReLU activation
- MaxPooling layers
- Dense layers
- Softmax output layer

## Model Training
- Training using TensorFlow/Keras
- Adam optimizer
- Categorical Crossentropy loss
- Validation split during training

## Hyperparameter Tuning
- Number of convolutional layers
- Filter size and filter count
- Pooling size
- Batch size
- Number of epochs

## Regularization Techniques
- Dropout layers
- L2 Regularization
- Early Stopping

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
