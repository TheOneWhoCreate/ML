# Linear Regression Models

This project focuses on regression techniques to model relationships between variables.  
It includes Linear Regression, Ridge, and Lasso, highlighting the role of regularization.

## What's Covered

### Model Implementation
- Simple Linear Regression  
- Ridge Regression (L2 Regularization)  
- Lasso Regression (L1 Regularization)  

### Performance Evaluation
- Mean Absolute Error (MAE)  
- Mean Squared Error (MSE)  
- Root Mean Squared Error (RMSE)  
- R² Score  

### Regularization Analysis
- Effect of L1 (feature selection)  
- Effect of L2 (coefficient shrinkage)  
- Overfitting vs underfitting  

## Visualization
- Regression line fitting  
- Predicted vs actual values  
- Error comparison plots  

## What You Will Learn

- How regression models predict continuous values  
- Differences between Ridge and Lasso  
- Why regularization is important  
- How to evaluate regression models effectively
