# Artificial Neural Network (ANN) Classifier

This project implements Deep Learning models using Artificial Neural Networks (ANN) for classification tasks.  
It compares different ANN architectures and applies regularization techniques to improve performance and reduce overfitting.

---

# What's Covered

## Artificial Neural Network (ANN)
- Fully Connected Neural Network  
- Forward propagation and backpropagation  
- Multi-class classification using Softmax  

## Data Preprocessing
- Train-Test split  
- Feature normalization using MinMaxScaler  
- One-hot encoding of target labels  

## ANN Architecture Design
- Hidden layers and neurons  
- Activation functions:
  - ReLU  
  - Sigmoid  
  - Tanh  

## Model Training
- Training using TensorFlow/Keras  
- Validation during training  
- Accuracy and loss monitoring across epochs  

## Overfitting & Regularization
- Detection of overfitting  
- Dropout layers  
- L2 Regularization  
- Improved model generalization  

## Model Comparison
- Comparison of multiple ANN architectures  
- Effect of changing:
  - Number of hidden layers  
  - Number of neurons  
  - Activation functions  

## Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  
- Confusion Matrix
