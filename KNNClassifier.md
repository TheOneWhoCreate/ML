# K-Nearest Neighbors (KNN) Classifier

This project implements the K-Nearest Neighbors (KNN) algorithm for classification tasks.  
It explores how model performance varies with different values of *k* and demonstrates decision boundary visualization.

## What's Covered

### Model Implementation
- KNN classifier using scikit-learn  
- Distance-based classification approach  
- Training and testing on dataset  

### Hyperparameter Tuning
- Experimenting with different values of k (e.g., 3–15)  
- Selecting optimal k based on accuracy  

### Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  

## Visualization
- Accuracy vs k plot  
- Decision boundary (2D feature space)  
- Classification result comparison  

## What You Will Learn

- How KNN works as a lazy learning algorithm  
- Impact of k on bias and variance  
- Importance of distance metrics  
- How to visualize classification boundaries
