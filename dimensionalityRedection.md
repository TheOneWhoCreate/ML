# Dimensionality Reduction

This project demonstrates dimensionality reduction techniques used to simplify high-dimensional datasets while preserving important information.  
It focuses on feature transformation, variance analysis, and visualization of reduced feature spaces.

## What's Covered

### Understanding High-Dimensional Data
- Feature scaling and standardization
- Importance of dimensionality reduction
- Variance and feature dominance concepts

### Principal Component Analysis (PCA)
- Applying PCA using scikit-learn
- Explained variance ratio analysis
- Selecting optimal number of components
- Transforming data into lower-dimensional space
- 2D visualization of principal components

### t-SNE (t-Distributed Stochastic Neighbor Embedding)
- Non-linear dimensionality reduction
- Visualizing complex data clusters
- Comparing PCA vs t-SNE behavior

### Visualization
- Scatter plots of reduced dimensions
- Variance comparison plots
- Cluster visualization

## What You Will Learn

- Why dimensionality reduction is important in machine learning
- How PCA reduces dimensionality using variance maximization
- How to interpret explained variance ratio
- Difference between linear (PCA) and non-linear (t-SNE) methods
- How scaling affects PCA results
- How to visualize high-dimensional data effectively
