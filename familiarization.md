# Familiarization

This project is a hands-on introduction to the core Python libraries used in scientific computing and data analysis.   The notebook demonstrates practical implementation of numerical operations, statistics, optimization, and visualization.

## What's Covered

### NumPy
- Array creation and manipulation  
- Mathematical operations on arrays  
- Matrix operations  
- Basic linear algebra  

### Pandas
- DataFrame creation  
- Data indexing and slicing  
- Descriptive statistics  
- Basic data exploration  

### Statistics
- Mean, median, variance, standard deviation  
- Summary statistics  
- Data interpretation  

### SciPy
- Linear algebra functions  
- Optimization techniques  
- Distance metrics  

### Data Visualization
- Line plots  
- Scatter plots  
- Basic graphical analysis using Matplotlib  

## What You Will Learn

- How to perform numerical computation using NumPy  
- How to manipulate structured datasets using Pandas  
- How to compute and interpret statistical measures  
- How to apply optimization and distance functions using SciPy  
- How to visualize data effectively  
- How scientific Python libraries integrate in real-world data workflows
