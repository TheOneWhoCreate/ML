# Decision Tree & Random Forest Classifier

This project implements tree-based models for classification tasks.  
It compares Decision Tree and Random Forest to understand the benefits of ensemble learning.

## What's Covered

### Decision Tree
- Tree-based classification model  
- Splitting criteria (Gini/Entropy)  
- Overfitting behavior  

### Random Forest
- Ensemble of multiple decision trees  
- Bagging technique  
- Improved generalization  

### Hyperparameter Tuning
- Grid Search for optimal parameters  
- Model comparison  

### Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  
- ROC-AUC  

## Visualization
- Decision boundary plots  
- ROC curves  
- Feature importance graph  

## What You Will Learn

- How tree-based models make decisions  
- Why Decision Trees overfit  
- How Random Forest reduces variance  
- Importance of ensemble methods
